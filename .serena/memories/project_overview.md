# Project Overview
- MimicXml is a .NET 8 CLI that produces entrapment XML databases for proteomics workflows, optionally relying on a bundled `mimic.exe` to scramble FASTA data while preserving PTMs and truncation products.
- Solution layout: `MimicXml.sln` hosts the CLI (`MimicXml`), shared services (`Core`), and NUnit tests (`Test`), with assets such as `mimic.exe`, protease RNase TSVs, and `Test/**/TestData` copied as needed.
- Tech stack: C# with file-scoped namespaces, nullable/implicit usings, depends on `mzLib`, `CommandLineParser`, `Microsoft.Extensions.DependencyInjection`, and targets .NET 8 on 64-bit Windows and cross-platform runtimes.
- CI flow (Windows) restores, builds release, runs coverage-aware tests, and keeps analyzer warnings at zero; bundling ensures `mimic.exe` is deployed with the CLI.
