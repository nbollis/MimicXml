MimicXml is a .NET 8 CLI for generating entrapment XML databases for proteomics workflows. Solution contains Core (library/services), MimicXml (CLI entrypoint), and Test (NUnit). CLI can ingest target XML + optional entrapment FASTA, or run bundled Windows mimic.exe to generate entrapment FASTA, then writes *_Entrapment.xml and optional histograms. DI is centralized in MimicXml/AppHost.cs. Key service areas: IO readers/writers and file type detection, mimic process runner, entrapment grouping and modification assignment, histogram calculators. CI runs on windows-latest with restore/build/test + coverage upload.
